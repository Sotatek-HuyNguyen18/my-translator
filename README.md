# AI Document Translator

Công cụ dịch thuật tài liệu thông minh (Excel, Word, PPTX, PDF) sử dụng Gemini AI.

## Các tính năng:
- Dịch giữ nguyên định dạng file.
- Hỗ trợ đa định dạng: .xlsx, .docx, .pptx, .pdf.
- Tự động phân tích bối cảnh và thuật ngữ chuyên ngành.
- Giao diện hiện đại, dễ sử dụng.

## Cách chạy:
Ứng dụng được đóng gói bằng Docker và chạy trên Hugging Face Spaces.
